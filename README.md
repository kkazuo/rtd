# rtd: GTFS-realtime protobuf binary decoder

Decode GTFS-realtime protobuf binary and print to stdout.

# Usage

```
Decode GTFS-realtime Feed Message:

    rtd [options] [files...]

Options:
    -d    Read from standard input.
    -w    Diff each files. (at least 2 files required.)
```
